一、设计背景与目标（格式：宋体，小四，1.5倍行距；首行空2格，段后空一行）
（一）设计背景
简述机器人仿真控制在工业自动化、机器人教学等领域的应用价值，结合 ROS 平台的优势，说明本设计的实际意义。
随着人口老龄化、慢性病和意外伤害等情况的持续增长，越来越多的人步入了步行能力衰退甚至丧失的群体中。如此庞大的残老群体,需要大量轮椅等代步工具，然而传统轮椅有很多不足，如: (1) 功能单一，目前市面上的轮椅大多只具有行走的功能，无法自主实现较为复杂的避障操作;(2)残障人士操控困难，手动轮椅需要使用者依靠自己或他人的力量手动驱动轮椅，大大增加了使用者的生理负担;(3)附加了智能功能的轮椅，其操作复杂，轮椅速度设置不合适，对于认知能力较差的老年人不友好等。为此我们搭建多功能智能轮椅，解决残老人群的代步需求实现一辆可以与用户进行语音交互、具备自主导航、避障功能的智能轮椅，为他们提供安全监控,减少意外伤害。
（二）设计目标 
本作业设置 5 个基础模块、1 个选做模块，以 2 人小组协作完成。
模块 1：ROS 开发环境配置与图像采集
搭建标准 ROS 工作空间，完成 Ubuntu+ROS 环境配置，部署 usb_cam、SLAM、Navigation 等功能包，可通过 rqt、RViz 正常监控数据。
编写 ROS 节点实现图像采集、发布与实时显示，完成话题通信验证。
实现至少 2 种图像预处理（灰度化、高斯滤波、直方图均衡化等），封装为独立 ROS 节点并完成数据收发验证。
模块 2：目标识别与特征提取
选定常见物体或几何图形为目标，基于 Hough 变换、ORB、Haar/HOG 等方法实现目标检测。
编写 ROS 识别节点，在 RViz 中完成可视化标注，。
分析光照、角度、背景对识别精度的影响，提出并实现优化方案。
模块 3：目标识别与追踪
基于 OpenCV Tracker 实现不少于 2 种跟踪算法（KCF、CSRT、光流法等）的对比实验。
实现识别与跟踪联动，实时输出目标坐标话题，绘制跟踪轨迹。
记录算法性能数据，完成速度、稳定性、抗干扰能力对比分析与参数优化。
模块 4：SLAM 地图构建与 Navigation 自主导航
采用 Gmapping 或 Cartographer 完成环境 SLAM 建图，地图无明显漂移、边界清晰。
配置 Navigation 导航包，完成 move_base、全局 / 局部路径规划、代价地图参数配置。
实现自主导航、避障、定点到达功能，支持 2D Nav Goal 指定目标点，鼓励实现目标跟随导航。
分析建图与导航误差来源，完成参数调优与改进。
模块 5：系统整合与工程化优化
编写 launch 文件实现全系统一键启动，完成模块间话题与服务通信联调。
保证系统连续稳定运行。
规范记录系统设计思路、模块接口、参数配置、调试过程、问题与解决方案。
选做模块：三维点云处理（加分项）
搭建 PCL+ROS 环境，实现点云数据采集、滤波、分割。
提取目标点云特征并在 RViz 中可视化，可与识别、跟踪模块联动。
不占用基础分值，完成可获得额外加分。
